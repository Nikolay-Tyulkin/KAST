## Target Device

- Board: `Waveshare ESP32-S3-LCD-1.69`
- Screen: integrated board LCD
- Controls: 3 external buttons
- Power: battery connected to the board
- Strap: standard 22 mm purchased band
- Main use case: row counting while knitting
