# Knitting Assistant (ESP-IDF MVP)

Прошивка для `Waveshare ESP32-S3-LCD-1.69` (без тачскрина):
- удерживает питание через `SYS_EN` после нажатия кнопки питания;
- показывает на LCD текущие ряды, таймер сессии, статус и заряд батареи в процентах;
- `+`/`-` кнопки увеличивают/уменьшают счетчик рядов только в активной сессии;
- универсальная кнопка запускает, ставит на паузу, продолжает и завершает сессию;
- завершенные сессии сохраняются в локальную историю NVS;
- длинное нажатие кнопки питания сохраняет текущую сессию и отключает latch (`SYS_EN=0`).

## Важно по пинам

Используются пины Waveshare ESP32-S3-LCD-1.69 V2:
- LCD: SCLK=6, MOSI=7, RST=8, DC=4, CS=5, BL=15
- PWR: `SYS_OUT=GPIO40`, `SYS_EN=GPIO41`
- Battery ADC: `GPIO1` (`ADC_UNIT_1`, `ADC_CHANNEL_0`), делитель `3`
- Кнопка `+`: `GPIO2` -> кнопка -> `GND`
- Кнопка `-`: `GPIO16` -> кнопка -> `GND`
- Универсальная кнопка: `GPIO17` -> кнопка -> `GND`

Кнопки работают с внутренней подтяжкой к питанию: отпущено `1`, нажато `0`.

## Сборка и прошивка (COM14)

Перед запуском команд активируйте ESP-IDF PowerShell профиль:

```powershell
. "C:\Espressif\tools\Microsoft.v6.0.1.PowerShell_profile.ps1"
```

Дальше в папке проекта:

```powershell
idf.py set-target esp32s3
idf.py -p COM14 build flash monitor
```

Выход из монитора: `Ctrl+]`
